### Описание проекта    
Угадать загаданное компьютером число за минимальное число попыток.

:arrow_up:[к оглавлению](https://github.com/SalnikovaMaria/My_projects#%D0%BE%D0%B3%D0%BB%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5)


### Какой кейс решаем?    
Нужно написать программу, которая угадывает число менее, чем за 20 попыток

**Условия соревнования:**  
- Компьютер загадывает целое число от 0 до 100, и нам его нужно угадать. Под «угадать», подразумевается «написать программу, которая угадывает число».
- Алгоритм сравниваем загаданное число со средним в последовательности от минимального до максимального числа, в случае неравенства значение среднего числа меняется на минимальное или максимальное и цикл повторяется.

**Метрика качества**     
Результаты оцениваются по количеству попыток при 1000 повторений

**Что практикуем**     
Учимся писать хороший код на Python.
